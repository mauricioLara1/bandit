## Sobre este repo

Cada writeup documenta el razonamiento y los comandos usados para resolver cada nivel — no solo la respuesta final. La idea es que sirva tanto de referencia personal como de muestra de progreso para quien lo revise (reclutadores, otros estudiantes de seguridad, etc.).

**Nota:** las contraseñas reales obtenidas en cada nivel no se incluyen en este repo público (ver `.gitignore`). Los writeups documentan el método, no las credenciales.
